​ 好久没更新了，但学了不少东西，今天写的是有关自动化脚本神器~~~autohotkey:

在我们平常经常要对pc进行重复的操作，比如打开百度，打开我们喜欢的电影网站等等。其实这些我们都可以通过一个简单的脚本来实现，那就不得不提到这个非常好用的自动化脚本软件–autohotkey。那下面就来说说它的一些语法：

下载好后，就可以随意创建一个文本文件，然后后缀改为.ahk,接着打开编辑。

先介绍一些基础的修饰符:

通过上面的修饰符加上自己选择要进行操作的字符或者字符串，就形成了一个热键，热键也称为快捷键，因为他们可以简便地触发动作，但在修饰符和字符串之间还要加上这个::

当我们想在一个热键里实现多条命令时，这时字符串就要放在::下，并在最后加上return,例如：

#n::

Run ~~~

Run***

return

当然如果你的热键设置了很多的时候，那也可以使用多个修饰符，只要将它们连续的列出来（顺序无关紧要），例如：

#^!s:: Send Sincerely,{enter}John Smith ; 此行发送键击到活动 (最前面的) 窗口

表示要通过按住windows+ctrl+alt+S触发热键

或者使用&定义组合键：按住 Numpad0 接着按下第二个按键时会触发此热键

多个热键可以垂直叠放来让它们执行相同的操作，例如：
